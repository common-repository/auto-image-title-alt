Automatically add title and alt tags to any new image uploaded to you the media library. This will improve your SEO and save you time while following the accsessibility guidelines. This light plugin will not affect your site performance at all.

== Description ==

**This plugin will improve your SEO by adding automatically the Title and Alt Text tags to any new image uploaded to you the media library. It will use the name of your file as the meta title and Alt Text, and will clean the name from any non-alphanumerical characters.**

This is a repeated manual task that needs to be done everytime a new image is uploaded and it can be 100% automated with this plugin.

It is super easy to use since there are no options that needs to be set. You just need to install the plugin and let the maginc happen.

**Plugin features**
- Add automatically the image title based on the file name.
- Add automatically the image Alt Text based on the file name.
- Improve your SEO giving Google more context about your images.
. Improve you website accessibility.
- Save time automating manual processes.
- Easy to use plugin. Just need to install and activate.
- Don't affect your site speed.
